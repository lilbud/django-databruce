Advanced Event Search

Search for events based on a number of parameters like month/day, who was onstage, and even what songs were played. I referenced the advanced search created by the folks over at Jerrybase during the creation of mine. It was a big help in figuring out the basic layout, and some of the extra code needed to make this work exactly like I wanted.
Start/Last Date accepts full dates as well as partial dates. So, entering '1975-02' for start will find events after February 1st, 1975. Likewise, entering '1975-02' for last will find events from before February 28th, 1975.
