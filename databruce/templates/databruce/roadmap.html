Roadmap

This page is to list things I plan to do on the site at some point. None of these have any kind of ETA, and many are simply just ideas I had that I'd like to do, but haven't sorted out the details.

Data Improvements

More detailed data/info, some ideas include:

  • Venue capacity?
  • Ticket sales per show

More Event Notes

Recently, a "notes" tab was added to the event detail page, clicking it shows nothing as of now. The loose plan for this is to host notes/reviews/articles pertaining to each show taken from various sources. Those include: Backstreets Mag/Point Blank (via my archive), BTX Article Dump, and the E Street Shuffle archive. Additionally, including archived links to various reviews as well for those I haven't pulled into a database of some kind.

Publicly Available Database

Right now, while the site is public, the database really isn't. I hope to make the database itself publicly accessible, likely in the form of a PostgreSQL database dump. This would be for those who do know SQL and want to mess around with a local copy of the database.

An API

This one is basically just a vague idea, as I don't have a clue how to do it in a "safe" way. Meaning how to allow public access to the API without affecting the site itself, and without jacking up my hosting costs as I'm sure it would have an impact.
